How to Play?
  • You can set their names by clicking on the default names(Player 1 or Player 2) and can set the winning score to any number by clicking on the "Winning Score" text before starting the game.
  • Changing the player names and the winning score is only possible before starting to play
  • Default winning score will be 100. First player reach to the winning score wins.
  • Each player will get a chance to roll two dices at the same time. The process given below will continues until one player reaches the winning score.
  • If 02 dices give the same result (except the dice value 1), the same player will get another chance to roll the dice. Else other player gets the chance to roll 02 dices.
  • When calculating the score, If both dices give the value 1, individual marks of the respective player will be reduced to 0. Else sum of the two dice values will be added to individual scores of the player
  • To play again, click on New Game
